About Visual_SLG
Visual_SLG v1.0 — Secondary Line visualizer

Born from one too many Microsoft Paint demos in Teams.

Quick guide
  • Trips: drag from the palette onto the grid; drag to move, drag the right edge to resize, click to recolor / set deadhead / delete (Delete).
  • Undo: Ctrl/⌘+Z.
  • Bid month: click a day header to mark the start; days left of the line are carry-over (no credit).
  • Trip Bank: set counts to stage bars, drag them onto the grid and back.
  • Randomizers: set Min/Max CH, then Build domestic or international.
  • Annotate: notes, arrows, pins, pen; Scenario panel for write-ups.
